# 💜 Финансовая диагностика барьеров

**[→ Открыть квиз](https://YOUR_USERNAME.github.io/income-barrier-quiz)**

## О проекте

Интерактивный опросник для определения главного психологического или стратегического барьера в росте дохода. Чистый HTML/CSS/JS — без фреймворков, работает прямо в браузере.

## Стек

- Vanilla JS (без зависимостей)
- Canvas API — анимированный aurora-фон и частицы
- Google Fonts: Cormorant Garamond + Manrope
- html2canvas — скачивание результата как изображения

## Барьеры

| Тип | Описание |
|-----|----------|
| 🧠 Психологический | Страх, самосаботаж, обесценивание себя |
| 🗺️ Стратегия | Нет системы, хаотичный доход |
| 🔧 Экспертиза | Нехватка конкретных инструментов |
| ⚡ Действие | Прокрастинация, откладывание |

## Запуск

Просто открой `index.html` в браузере — никаких сборщиков, никаких `npm install`.

1. Settings → Pages → Source: `main` branch, `/ (root)`
